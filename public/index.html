How it works

Reverse alpha blending

Gemini applies its watermark using a known alpha compositing formula. We simply invert it.

01
Detect watermark location
Based on image dimensions, the engine determines whether the 48×48 or 96×96 watermark template applies and precisely calculates its bottom-right position.
02
Build the alpha map
A precomputed watermark template is loaded and converted to a per-pixel alpha (opacity) map. Each value tells us how strongly the watermark affected that pixel.
03
Recover original pixels
For each watermarked pixel, the formula C = (C_w − α·255) / (1 − α) reconstructs the original color. Pixels below the alpha threshold are left untouched.
04
Export as lossless PNG
The restored ImageData is written back to a canvas and exported at full resolution. Zero recompression, zero quality loss.
Watermark compositing model
Cw = α · Clogo + (1 − α) · Corig
Inverse — recover original pixel
Corig = (Cw − α · 255) / (1 − α)
Constants
αmin = 0.002 · αmax = 0.99
Clogo = 255 (white watermark)
Sizes: 48×48 · 96×96
